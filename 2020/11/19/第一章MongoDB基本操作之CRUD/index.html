2.1 db.collection.find()

语法：db.<collection>.find(<query>, <projection>)
<query>文档定义了读取操作时筛选文档的条件
<projection>文档定义了对读取结果进行的投射

2.2 筛选文档 比较操作符

{ <field>: { <operator>: <value> } }

  • $eq 匹配字段值相等的文档

  • $ne 匹配字段值不等的文档

  • $gt 匹配字段值大于查询值的文档

  • $gte 匹配字段值大于或等于查询值的文档

  • $lt 匹配字段值小于查询值的文档

  • $lte 匹配字段值小于或等于查询值的文档

    另外两个比较操作符

  • $in

  • $nin

    逻辑操作符

  • $not 匹配筛选条件不成立的文档

  • $and 匹配多个筛选条件全部成立的文档

  • $or 匹配至少一个筛选条件成立的文档

  • $nor 匹配多个筛选条件全部不成立的文档

    $not

    { field: { $not: { <operator-expression> } } }

    $and

    { $and: [ {<expression1>}, {<expression1>}, {<expression1>},] }

    $or 和 $nor不加赘述，自行脑补

    字段操作符

  • $exists 匹配包含查询字段的文档

  • $type 匹配字段类型符合查询值的文档

    $exists

    表达式: { field: { $exists: <boolean> } }

    $type
