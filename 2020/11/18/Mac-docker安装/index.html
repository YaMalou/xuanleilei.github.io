Mac docker安装 及初步安装MongoDB以及mongo express

一、Docker for Mac 下载

  • 下载地址docker
  • 安装完成后注册docker id进行登录
  • 命令行运行docker –version查看是否安装成功

二、下载MongoDB官方docker镜像

查看下载镜像

启动一个MongoDB服务器容器

  • –name mymongo –>容器名字
  • -v /mymongo/data:/data/db –>挂载数据目录

如果出现/mymongo/data无法访问的问题，可以去docker Desktop –> Preferences –> Resources –> FILE SHARING 中添加/mymongo/data目录即可

下载mongo-express镜像

运行mongo-express
